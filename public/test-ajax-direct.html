3. Guide pour remplir les champs

ConfigId:

  1. Ouvre Step 5 dans le navigateur
  2. Regarde l'URL: http://localhost:8000/...bullet-ng/ICI/step5
  3. Copie la valeur entre `/bulletin-ng/` et `/step5`

CSRF:

  1. Ouvre F12 (Console)
  2. Tape: document.querySelector('meta[name=csrf-token]').content
  3. Copie le résultat

StudentId et SubjectId:

  1. Dans la page Step 5, clique droit sur un input de note
  2. "Inspecter l'élément"
  3. Regarde l'attribut data-student et data-subject
  4. Copie les valeurs
